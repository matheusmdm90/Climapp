# Projeto de Previsão de Clima 🌤️

## Introdução:
Esse é um projeto de previsão de clima onde quis colocar em prática meus conhecimentos em React Native utilizando o Expo e consumindo APIs externas, como o OpenWeather. O layout foi desenvolvido para funcionar tanto em Android quanto em iOS, e acredito que consegui aplicar bem esses conceitos.

## Resumo do App:
Aplicativo de previsão do tempo, onde é possível ver a previsão atual para as principais cidades do Brasil, utilizando o React Native com Expo e algumas bibliotecas próprias do Expo.

## Código do Projeto:
Você pode visualizar o código do meu projeto no GitHub ou até mesmo baixar o projeto para o seu computador. Deixei alguns trechos do código explicados para facilitar o entendimento.

OBS: Fique à vontade para modificar o projeto e até sugerir melhorias. Será um prazer receber suas observações.

## Instalar as Dependências:
1. `npm install`
2. Inicie o aplicativo com:
    `npx expo start`

## Expo GO:
